This repository contains applications that demonstrate the shuffling of a deck of cards, using the Fisher-Yates Shuffle algorithm (https://en.wikipedia.org/wiki/Fisher%E2%80%93Yates_shuffle)

The applications are:

C Shuffling
- A C representation of a deck shuffling application
- First Version: March 24th, 2020
- Updated on March 25th, 2020 based on a code review by Mark Paruzel (https://github.com/mparuzel).

C Sharp Shuffling
- An application, written in C#, that using Object Oriented principles to demonstrate
- First Version: March 23rd, 2020
